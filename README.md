# FirstProject# Banking App

A simple C# console banking application built to practice OOP, project structure, validation, and transaction history.

## Features
- Create bank accounts
- Select account by account number
- Deposit money
- Withdraw money
- View balance
- View transaction history
- List all accounts

## How to run
1. Clone the repo
2. Open `FirstProject.sln` in Visual Studio
3. Run the project

## Tech
- C#
- .NET Framework / Console App
